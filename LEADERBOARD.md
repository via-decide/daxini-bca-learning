## 🚀 How to Submit Your Project

Have you successfully built and hosted a project from this repository? Add it to the leaderboard!

1. **Fork this repository.**
2. **Edit this file (`LEADERBOARD.md`).** Add your project to the bottom of the "Community Submissions" table.
3. **Make a Pull Request (PR)** with the title: `Add [Your Name]'s [Project Name] to Leaderboard`.

### What you need to provide:
- **Project / Domain:** Which project you built (e.g., URL Shortener, Gym Management).
- **Student:** Your GitHub username / real name.
- **Notes:** Any cool custom features you added (e.g., "Added dark mode" or "Used PostgreSQL instead of SQLite").
- **Links:** A link to your Live Demo (Vercel/Render) AND a link to your GitHub source code.
