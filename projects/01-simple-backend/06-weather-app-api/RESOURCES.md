# ⛅ Weather App API: Learn By Building

**"Build a proxy API that fetches real-time weather from a third-party service, caches the results to save money and improve speed, and returns a clean, customized JSON response to your frontend."**

---

## 📚 Resources

**Third-Party APIs:**
- OpenWeatherMap Current Weather Data (Free Tier): https://openweathermap.org/current

**Backend HTTP Requests:**
- Axios (Popular HTTP client for Node.js): https://axios-http.com/docs/intro
- Node `fetch` API (Built-in for modern Node.js): https://nodejs.org/api/globals.html#fetch

**Caching & Databases:**
- Caching Strategies (A great conceptual read): https://aws.amazon.com/caching/best-practices/
- SQLite `UPSERT` (INSERT ON CONFLICT): https://sqlite.org/lang_UPSERT.html

**Design Patterns:**
- The Backend-For-Frontend (BFF) Pattern: https://samnewman.io/patterns/architectural/bff/
- The Adapter Pattern: https://refactoring.guru/design-patterns/adapter
