# 💱 Currency Converter API: Learn By Building

**"Build a fast, precise API that converts between global currencies by fetching and caching live exchange rates."**

---

## 📚 Resources

**3rd Party APIs:**
- ExchangeRate-API (Free tier, reliable): https://www.exchangerate-api.com/
- Frankfurter (Open-source, no API key required, based on European Central Bank): https://www.frankfurter.app/

**Financial Math in JavaScript:**
- Why `0.1 + 0.2 = 0.30000000000000004`: https://0.30000000000000004.com/
- `decimal.js` (The standard library for fixing JS math): https://github.com/MikeMcl/decimal.js/
- `currency.js` (Alternative library): https://github.com/scottyzen/currency.js

**Caching Patterns:**
- Fallback caching (Returning stale data if the upstream is dead): https://docs.microsoft.com/en-us/azure/architecture/patterns/cache-aside
