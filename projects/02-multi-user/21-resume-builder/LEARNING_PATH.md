## 🎯 Learning Outcomes
✅ **JSONB in SQL** - Knowing when to break normalization rules for document-oriented data.
✅ **PDF Generation** - Passing HTML/CSS to a headless browser (like Puppeteer) to generate a PDF.
✅ **Draft States** - Saving auto-drafts of complex forms.

## 📋 Project Overview
Build a tool where users fill out a dynamic form with their work history, and the system renders it into a beautifully formatted, downloadable PDF.
