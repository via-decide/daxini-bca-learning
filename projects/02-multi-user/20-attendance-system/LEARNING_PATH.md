## 🎯 Learning Outcomes
✅ **Interval Calculation** - Extracting hours and minutes from two timestamps.
✅ **Graceful Fallbacks** - Handling the "User forgot to punch out" scenario via automated scripts.
✅ **Idempotency** - Preventing double punches.

## 📋 Project Overview
Build a digital punch clock where employees log their arrival and departure times, and the system aggregates daily and weekly hours worked.
