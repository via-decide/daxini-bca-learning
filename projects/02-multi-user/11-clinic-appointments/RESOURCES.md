# 🩺 Clinic Appointments API: Learn By Building

**"Build a scheduling API where Doctors define their weekly availability, and Patients book specific 15-minute time slots, requiring strict time-boundary logic and overlap prevention."**

---

## 📚 Resources

**Time Math in Javascript:**
- You will need to add 15 minutes to a time string. While you can write this manually, using a library makes it much safer.
- *Date-fns* (Modern, lightweight alternative to Moment.js): https://date-fns.org/
  - Specifically look at `addMinutes` and `parse`.

**Database Constraints:**
- Composite Unique Keys (Preventing double bookings): https://www.w3schools.com/sql/sql_unique.asp
