## 🎯 Learning Outcomes
✅ **Unique Constraints** - Using DB constraints to enforce business rules natively.
✅ **Time Slots** - Handling fixed interval scheduling (e.g. 30-minute blocks).
✅ **Role-Based Views** - Doctors see their schedule, patients see their history.

## 📋 Project Overview
Build a booking portal where patients can view a doctor's availability and reserve a specific time slot, with strict prevention of double bookings.
