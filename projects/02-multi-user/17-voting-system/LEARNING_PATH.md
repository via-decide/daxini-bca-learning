## 🎯 Learning Outcomes
✅ **Atomic Operations** - Updating counters safely under heavy load.
✅ **Idempotency via DB Constraints** - Leaning on the database to enforce rules, not the application logic.
✅ **High Concurrency** - Understanding why basic CRUD breaks when 10,000 people click a button per second.

## 📋 Project Overview
Build a real-time polling application where users cast votes, and the results update securely without dropping or duplicating a single vote.
