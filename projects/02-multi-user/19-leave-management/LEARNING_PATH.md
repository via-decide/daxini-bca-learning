## 🎯 Learning Outcomes
✅ **Date Logic** - Excluding weekends when calculating `days_requested` between two dates.
✅ **Quota Management** - Safely decrementing balances only upon approval.
✅ **Approval Workflows** - State transitions (Pending -> Approved).

## 📋 Project Overview
Build an HR tool where staff request time off. The system automatically computes the required days (skipping weekends) and subtracts them from annual quotas.
