## 🎯 Learning Outcomes
✅ **Hierarchical Data** - Managing deep relationships (Project -> Sprint -> Task).
✅ **Graph Data Models** - Implementing Task Dependencies (Directed Acyclic Graphs).
✅ **Sprint Logic** - Time-boxing tasks and rolling over unfinished work.

## 📋 Project Overview
Build a project tracker where managers can plan 2-week Sprints, developers can pick up tickets, and the system physically blocks moving a task to "Done" if its dependent tasks aren't completed yet.
