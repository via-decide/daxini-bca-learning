## 🎯 Learning Outcomes
✅ **Inventory States** - Determining if an item is available based on transactional history.
✅ **Date Differencing** - Calculating late fees based on calendar math.
✅ **Data Consistency** - You cannot return a book that isn't currently borrowed.

## 📋 Project Overview
Build a digital library where patrons check out books, librarians manage stock, and the system automatically enforces late penalties.
