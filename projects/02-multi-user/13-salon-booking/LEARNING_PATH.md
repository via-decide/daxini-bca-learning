## 🎯 Learning Outcomes
✅ **Dynamic Range Overlaps** - Preventing booking collisions when durations are variable.
✅ **Computed Columns / Application Math** - Determining `end_time` before insertion.
✅ **Calendar Architectures** - Thinking in timelines instead of slots.

## 📋 Project Overview
Build a booking system for a hair salon where customers select a specific service (with a predefined duration) and the system finds valid gaps in a stylist's schedule.
