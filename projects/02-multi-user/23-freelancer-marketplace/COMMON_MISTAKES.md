## ⚠️ Common Mistakes

❌ **Mistake 1: Accepting multiple bids**
If a client double-clicks the "Accept" button on two different bids, you might award the project to two freelancers. Use a transaction: check if the project status is still 'open' before updating the bid to accepted.

❌ **Mistake 2: Milestone Math**
The sum of all Milestones on a project should equal the accepted Bid amount. Your API must validate this before allowing the contract to begin.
