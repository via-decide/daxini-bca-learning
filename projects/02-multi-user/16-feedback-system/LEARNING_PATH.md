## 🎯 Learning Outcomes
✅ **Aggregations** - Calculating `AVG()`, `COUNT()`, and grouping data.
✅ **Database Constraints** - Enforcing `CHECK (rating >= 1 AND rating <= 5)`.
✅ **Caching** - Knowing when to compute averages on the fly vs storing a cached total.

## 📋 Project Overview
Build a universal feedback engine where users can submit reviews, and the backend aggregates scores to build leaderboards or product rating pages.
