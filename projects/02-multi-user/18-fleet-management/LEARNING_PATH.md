## 🎯 Learning Outcomes
✅ **Time-Series Data** - Handling massive amounts of append-only IoT data.
✅ **State Separation** - Why updating the current state is different from logging the history.
✅ **Performance Tuning** - Realizing that writing to `Location_Telemetry` 500 times a second requires batching or specialized databases.

## 📋 Project Overview
Build a backend that accepts GPS pings from hardware devices, updates a live map of all vehicles, and allows managers to replay a truck's route from yesterday.
