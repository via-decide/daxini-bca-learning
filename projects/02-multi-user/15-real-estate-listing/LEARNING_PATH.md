## 🎯 Learning Outcomes
✅ **Geospatial Queries** - Finding items within a radius using coordinates.
✅ **One-to-Many Media** - Associating multiple images with a single entity.
✅ **Sorting Algorithms** - Sorting by distance vs sorting by price.

## 📋 Project Overview
Build a property portal where agents upload houses (with image galleries) and buyers search for homes nearby using map coordinates.
