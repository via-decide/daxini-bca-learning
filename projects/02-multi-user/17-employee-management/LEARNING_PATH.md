## 🎯 Learning Outcomes
✅ **Lookup Tables** - Normalizing categorical data to prevent typos and enable easy renaming.
✅ **Soft Deletes** - Changing status to `Terminated` instead of physically deleting the row.
✅ **Joins** - Combining 3 tables to generate a single readable profile.

## 📋 Project Overview
Build a core HR database to onboard new hires, assign them to departments, and track their employment lifecycle.
