## 🎯 Learning Outcomes
✅ **Event Sourcing (Basic)** - Storing a ledger of changes rather than just the current state.
✅ **Audit Trails** - Tracking accountability for destructive actions.
✅ **Trigger Thresholds** - Emitting alerts when computed values drop below limits.

## 📋 Project Overview
Build a warehouse system where staff scan items in and out, the system prevents stock from going below zero, and managers can audit exactly where missing stock went.
