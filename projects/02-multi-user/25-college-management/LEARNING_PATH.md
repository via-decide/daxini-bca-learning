## 🎯 Learning Outcomes
✅ **Deep Hierarchy** - Navigating nested relationships (Course -> Assignment -> Submission -> Student).
✅ **Access Control** - A student can only view assignments for courses they are enrolled in.
✅ **Aggregation** - Calculating a final grade based on a weighted average of all submission scores.

## 📋 Project Overview
Build a Learning Management System where faculty create syllabi and assignments, and students submit work for grading and track their academic progress.
