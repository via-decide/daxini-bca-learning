## 🎯 Learning Outcomes
✅ **Server-Side Security** - Never trust the frontend timer.
✅ **Idempotency** - Ensuring a double-click doesn't submit a test twice.
✅ **Automated Grading** - Joining student answers with the correct answers to compute a score dynamically.

## 📋 Project Overview
Build a test portal where teachers create quizzes and students take them. The system strictly enforces time limits and auto-grades the submissions instantly.
