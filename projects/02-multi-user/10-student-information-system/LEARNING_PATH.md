## 🎯 Learning Outcomes
✅ **Many-to-Many Relationships** - The most common SQL pattern.
✅ **Derived Data** - Why you shouldn't store the `GPA` in the database, but calculate it on the fly.
✅ **Academic Integrity** - Preventing grade changes after a semester is locked.

## 📋 Project Overview
Build an SIS where admins create courses, students enroll in them, and professors assign grades. The system calculates transcripts automatically.
