## 🎯 Learning Outcomes
✅ **Billing Cycles** - Designing schemas that handle recurring monthly charges.
✅ **Invoice vs Payment** - Separating the concept of "What is owed" from "What was paid".
✅ **Partial Payments** - Handling scenarios where a tenant pays $500 towards a $1000 invoice.

## 📋 Project Overview
Build a property management dashboard where landlords track their leases, generate monthly rent invoices, and log tenant payments.
