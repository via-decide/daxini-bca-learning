## 🎯 Learning Outcomes
✅ **State Machines** - Enforcing valid transitions (e.g., you can't move a ticket from "Closed" to "In Progress" without a new comment).
✅ **Polymorphic-ish Relations** - The `user_id` in comments can belong to different roles.
✅ **Internal Notes** - Masking certain comments from the customer's view.

## 📋 Project Overview
Build a helpdesk where users submit issues, and support agents claim, discuss, and resolve those issues through a shared dashboard.
