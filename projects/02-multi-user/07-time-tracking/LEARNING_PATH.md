## 🎯 Learning Outcomes
✅ **Time Interval Math** - Calculating durations between two timestamps safely.
✅ **State Management** - Handling "running" timers vs "completed" timers.
✅ **Aggregation Queries** - Summing up total billable hours per client per month.

## 📋 Project Overview
Build a timesheet app where users log hours against specific projects, and managers can generate invoices based on the total accumulated time at specific hourly rates.
