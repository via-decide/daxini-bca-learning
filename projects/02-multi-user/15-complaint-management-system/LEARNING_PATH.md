## 🎯 Learning Outcomes
✅ **State Transition Auditing** - Keeping a historical log of status changes for accountability.
✅ **Geo-Tagging** - Storing location data alongside user reports.
✅ **File Attachments** - Handling multi-part form data for image uploads.

## 📋 Project Overview
Build a civic tech portal where citizens submit geo-tagged complaints with photos, and administrators manage the resolution workflow while keeping a transparent audit trail.
