## 🎯 Learning Outcomes
✅ **Server-Sent Events (SSE)** - One-way real-time data flow from server to client.
✅ **Pub/Sub Pattern** - Using Redis (optional) to publish events that background workers can subscribe to.
✅ **Unread Counts** - Efficiently querying and caching the number of unread alerts.
