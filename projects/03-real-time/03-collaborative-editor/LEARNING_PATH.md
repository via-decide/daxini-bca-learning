## 🎯 Learning Outcomes
✅ **Conflict Resolution** - Handling race conditions when multiple users edit state.
✅ **Delta Updates** - Sending only what changed (a patch/diff) rather than the entire 10,000-word document on every keystroke.
✅ **Cursor Tracking** - Broadcasting a user's cursor position to others in real-time.

## 📋 Project Overview
Build a real-time text editor where multiple people can type on the same page, see each other's cursors, and avoid overwriting each other's work.\n
