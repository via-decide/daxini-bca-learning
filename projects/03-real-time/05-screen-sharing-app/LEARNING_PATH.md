## 🎯 Learning Outcomes
✅ **WebRTC Protocol** - Understanding peer-to-peer data streams in the browser.
✅ **Signaling Servers** - Exchanging SDP (Session Description Protocol) and ICE candidates via WebSockets.
✅ **Browser Media APIs** - Using `navigator.mediaDevices.getDisplayMedia()` to capture the screen.

## 📋 Project Overview
Build an application where one user can share their screen, and another user can connect via a link and watch the live stream with sub-second latency.\n
