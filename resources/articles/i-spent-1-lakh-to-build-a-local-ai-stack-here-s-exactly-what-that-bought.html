That's it.

That's the total hardware investment behind the Daxini Stack — a local-first AI reasoning engine, a RAG system over 12,000 technical books, an async orchestration layer, and a published model on Ollama.

The majority of that ₹1 lakh went to one machine.

Mac Mini M4. Base variant. 16GB RAM. 256GB storage.

Not the pro version. Not a GPU server. Not a cloud instance billed by the hour.

When I pushed Zayvora — a 6-stage reasoning engine built on quantized Llama 3 — from local training to Ollama, the Mac Mini took 30 to 40 minutes.

It lagged the whole time.

It never failed once.

The M4 Neural Engine handled the entire process like it was killing a mosquito. Apple spent billions engineering that chip. I spent ₹1 lakh and got the benefit of all of it.

Zayvora — a reasoning engine running a 6-stage loop before producing any output. Decompose → Retrieve → Synthesize → Calculate → Verify → Revise. If verification fails, it produces nothing until resolved. If evidence is insufficient, it says clearly: "I don't know, and here's exactly why."

Nex — hybrid RAG over 12,000+ technical books locally. Dense embeddings + BM25 + Reciprocal Rank Fusion. ~200ms query time. Nothing leaves the machine.

Orchade — asyncio orchestration layer. Job queuing, priority scheduling, resource budgets.

StudyOS — single HTML file interface. No framework. No build step.

Two books on Amazon KDP. One model live on Ollama. One GitHub repo.

All of it. ₹1 lakh. 3 months. One machine in Gandhidham, Kutch.
