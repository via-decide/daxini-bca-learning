# SSL TLS

## Purpose
This page specifies the SSL/TLS chapter for Module 7: Web Infrastructure.

## Concepts
- **TLS**: Transport encryption for web traffic.
- **Certificates**: Identity documents issued by certificate authorities.
- **HTTPS**: HTTP carried over TLS.
- **Certificate renewal**: Operational process that prevents avoidable outages.
