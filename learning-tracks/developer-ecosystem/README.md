# Modern Software Engineering Knowledge Base

This learning track bridges academic BCA subjects with the vocabulary, tools, workflows, and production practices used by professional software engineers. It does not replace the existing curriculum; it adds an industry-facing reference layer.

## How to Use This Track

1. Read one module at a time.
2. Keep the [Engineering Glossary](./glossary.md) open and search terms as they appear.
3. Connect each tool name to the underlying concept.
4. Practice with small projects before using a tool in production.
