# ⌨️ Keyboard-Only Development (Ditch the Mouse)

Watch any senior developer work, and you will notice something peculiar: **they rarely touch their mouse.** 

Every time you move your hand from the keyboard to the mouse, you lose milliseconds. Over a day, that translates to minutes. Over a career, it translates to hundreds of hours. More importantly, using a mouse breaks your "flow state" and focus.

Adapting to a keyboard-only workflow is a superpower, but it takes time. Do not try to memorize everything at once. Pick 2-3 shortcuts this week, force yourself to use them until they become muscle memory, and then learn 2 more.

Here is the roadmap to becoming a keyboard ninja.

---

## 🟢 Level 1: Operating System Navigation
*Stop clicking on the taskbar/dock to switch applications.*

### Switching Applications
- **Windows/Linux:** `Alt + Tab`
- **macOS:** `Cmd + Tab`
*(Hold the first key and repeatedly press Tab to cycle through open apps. Add `Shift` to go backward).*

### Closing the Current Window
- **Windows/Linux:** `Alt + F4` or `Ctrl + W`
- **macOS:** `Cmd + Q` (quit app) or `Cmd + W` (close window)

### Opening Applications Instantly
Stop clicking the start menu. Use your system's global search to launch apps instantly.
- **Windows:** Press the `Windows Key`, type the app name (e.g., "vs"), hit `Enter`.
- **macOS:** Press `Cmd + Space` (Spotlight), type the app name, hit `Enter`.

---

## 🟡 Level 2: Browser Mastery (Chrome/Brave/Edge)
*Stop clicking the "+" icon to open new tabs or clicking the back arrow.*

### Tab Management
- **Open New Tab:** `Ctrl + T` (Win/Lin) | `Cmd + T` (Mac)
- **Close Current Tab:** `Ctrl + W` (Win/Lin) | `Cmd + W` (Mac)
- **Reopen Closed Tab (Lifesaver!):** `Ctrl + Shift + T` (Win/Lin) | `Cmd + Shift + T` (Mac)
- **Switch Tabs:** `Ctrl + Tab` (moves right), `Ctrl + Shift + Tab` (moves left). On Mac, you can also use `Cmd + Option + Right/Left Arrows`.

### Navigation
- **Focus the URL/Address Bar:** `Ctrl + L` (Win/Lin) | `Cmd + L` (Mac). *Instantly starts typing a new URL or Google search.*
- **Go Back a Page:** `Alt + Left Arrow` (Win/Lin) | `Cmd + Left Arrow` (Mac)
- **Go Forward a Page:** `Alt + Right Arrow` (Win/Lin) | `Cmd + Right Arrow` (Mac)

---

## 🔴 Level 3: The Code Editor (VS Code / Cursor)
*Your editor is where you spend 90% of your time. Mastering these is non-negotiable.*

### Finding Things Instantly
- **Quick Open (Find a file by name):** `Ctrl + P` (Win/Lin) | `Cmd + P` (Mac)
- **Global Search (Find a word anywhere):** `Ctrl + Shift + F` (Win/Lin) | `Cmd + Shift + F` (Mac)
- **Go to specific Line Number:** `Ctrl + G` (Win/Lin) | `Ctrl + G` (Mac)

### Editing Code Like a Pro
- **Move an entire line of code up/down:** `Alt + Up/Down Arrows` (Win/Lin) | `Option + Up/Down Arrows` (Mac). *No more cut and paste!*
- **Copy line up/down:** `Shift + Alt + Up/Down` (Win/Lin) | `Shift + Option + Up/Down` (Mac).
- **Multiple Cursors (Change the same word in multiple places):** Highlight a word, then press `Ctrl + D` (Win/Lin) | `Cmd + D` (Mac) to select the next occurrence of that word. Type to change them all simultaneously.
- **Delete an entire line:** `Ctrl + Shift + K` (Win/Lin) | `Cmd + Shift + K` (Mac)

### Managing the Editor View
- **Toggle the Integrated Terminal:** `Ctrl + \`` (Win/Lin) | `Cmd + \`` (Mac)
- **Toggle the left Sidebar (Explorer):** `Ctrl + B` (Win/Lin) | `Cmd + B` (Mac)

---

## 💀 Level 4: Vim (The Final Boss)

Once you master Level 3, the final evolution of a keyboard-only developer is learning **Vim**.

Vim is an editor (or a plugin for VS Code) where you don't even use the arrow keys. You keep your fingers perfectly still on the "Home Row" and use `H`, `J`, `K`, and `L` to move left, down, up, and right.

It takes weeks to learn, but once you do, you edit text at the speed of thought. As a beginner, don't worry about Vim yet—but know that it exists as the ultimate endgame for keyboard ninjas.
